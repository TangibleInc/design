# Overview

The Design library provides building blocks and patterns for user interface design.

- Generic: Styles based on a set of design tokens for easy theming.
- Modular: Features can be loaded individually.
- Local styles and scripts: Prefixed CSS classes and data attributes. Suitable for use with WordPress plugins, frontend UI or admin screen, where there is already a global stylesheet.

The library started as a rewrite of React Bootstrap and Design Tokens CLI, with other open-source libraries and utilities.
