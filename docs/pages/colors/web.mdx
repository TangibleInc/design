# CSS Named Colors

The CSS language specification defines a set of named colors, which are keywords that represent colors in [sRGB space](https://en.wikipedia.org/wiki/SRGB). These are supported by all major browsers.

- [W3C - CSS Color Module Level 4: Color Keywords - Named Colors](https://drafts.csswg.org/css-color/#named-colors)
